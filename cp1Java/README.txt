Descrição
Este projeto em Java implementa um programa de criptografia simples que permite aos usuários criptografar e descriptografar texto usando dois métodos populares: a Cifra de César e o Código Morse.

Pré-requisitos
Certifique-se de ter o JDK (Java Development Kit) instalado em seu sistema para poder compilar e executar o código.

Detalhes da Implementação
O programa consiste em três classes principais:

Main.java
Esta é a classe principal que contém o método main. Ele interage com o usuário, solicitando o tipo de criptografia e a operação a ser realizada, e chama os métodos apropriados das classes Cesar e Morse.

Cesar.java
Esta classe implementa a criptografia e descriptografia usando a Cifra de César. Ele contém métodos para criptografar e descriptografar um texto com base em um deslocamento fixo de caracteres.

Morse.java
Esta classe implementa a criptografia e descriptografia usando o Código Morse. Ele converte caracteres alfabéticos e numéricos em sequências de pontos e traços de acordo com o padrão Morse.
